Q4 Write a code to calculate area of a rectangle using inner function. In this case outer function should accept parameter length and inner function should accept parameter breadth.


Ans            const calculateRectangleArea = (length) => (breadth) => length * breadth;

            const lengthOfRectangle = 5;
            const calculateAreaWithLength5 = calculateRectangleArea(lengthOfRectangle);
            
            const breadth1 = 10;
            const area1 = calculateAreaWithLength5(breadth1);
            console.log("Area 1:", area1);
            
            const breadth2 = 7;
            const area2 = calculateAreaWithLength5(breadth2);
            console.log("Area 2:", area2);

Q5 Take a variable in outer function and create an inner function to increase the counter every time it is called 


Ans 
        function createCounter() {
            let counter = 0;
          
            function increaseCounter() {
              counter++;
              console.log("Counter:", counter);
            }
          
            return increaseCounter;
          }
        
          const counterFunction = createCounter();
          
          counterFunction(); // Output: Counter: 1
          counterFunction(); // Output: Counter: 2
          counterFunction(); // Output: Counter: 3
